🌐 Text Field Translation Feature Test

How to test:

  1. Make sure the TextMind Pro extension is loaded and configured with an AI provider
  2. Type some text in any of the fields below
  3. Look for the 🌐 translation button that appears on the right side of the field
  4. Click the button or use the keyboard shortcut Ctrl+Shift+T
  5. Select a target language and click "Translate"
  6. Optionally click "Replace" to replace the original text with the translation
